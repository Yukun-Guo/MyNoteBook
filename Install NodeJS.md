# Install NodeJS

## Install NodeJS 18 on Ubuntu 20.04

```bash
sudo apt update

sudo apt upgrade

curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

sudo apt-get install -y nodejs

node --version
```
